$ npm i -g leanweb
$ mkdir hello && cd hello
$ lw init # scaffold a project
$ lw serve # dev server, watch & rebuild
lw init

Scaffold a ready-to-run project in an empty directory. Light DOM with scoped CSS by default; add --shadow-dom for full encapsulation.

lw generate login

Create a component — login.html, login.css, login.js — and register it in leanweb.json. Nested names like admin/users work too.

lw serve

Start the dev server and rebuild on every change. lw serve prod runs against the prod environment.

lw dist

Production build: bundled, minified, source-mapped, written to dist/. Runs your post-dist script when one exists.

lw upgrade

Refresh the project runtime after updating the tools. lw upgrade --diff previews every file it would change without writing anything.

lw help

Everything else: build, clean, addpage, destroy, version. Commands match by prefix — lw s, lw di, lw g all work.
