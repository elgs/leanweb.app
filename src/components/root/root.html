Directives

Bind data where it lives — in the DOM

A handful of lw- attributes cover text, conditions, lists, events, and two-way binding. Every expression is plain JavaScript evaluated against your component.

Forms

Every control, one directive

lw-model keeps form state and component state in sync — checkboxes, radios, selects, sliders, and values nested deep inside arrays.
