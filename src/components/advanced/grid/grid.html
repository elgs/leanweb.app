Nested loops & 2D arrays#

Loops nest freely. One subtlety: a loop variable holding a primitive is a copy, so bind through the array instead — lw-model='row[$i]' writes back by reference. The sum below recomputes on every keystroke.


            <div class='r' lw-for='row in board'>
               <input type='number' lw-for='cell, $i in row' lw-model='row[$i]'>
            </div>
            <p>Sum: <b lw>total()</b></p>
         

            class extends LWElement {
               board = [
                  [1, 2, 3],
                  [4, 5, 6],
               ];

               total() {
                  return this.board.flat().reduce((a, b) => a + b, 0);
               }
            }
