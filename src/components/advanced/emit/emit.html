Signals up: custom events#

A child notifies its parent by dispatching a CustomEvent; the parent listens with lw-on: right on the child's tag — same directive, your event name, payload on $event.detail. The listener belongs to the parent, so the parent re-renders.


            <p>Cart total <b lw>count</b></p>
            <demo-child lw-on:add='count += $event.detail'></demo-child>
         

            customElements.define('demo-parent',
               class extends LWElement {
                  count = 0;
               }
            );
         

            <button lw-on:click='send(1)'>Add one</button>
            <button lw-on:click='send(5)'>Add five</button>
         

            customElements.define('demo-child',
               class extends LWElement {
                  send(n) {
                     this.dispatchEvent(new CustomEvent('add', { detail: n }));
                  }
               }
            );
