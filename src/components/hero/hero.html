Web components.
No framework required.

Leanweb is a tiny toolchain that turns plain HTML, CSS, and JavaScript into reactive, standards-based custom elements. No virtual DOM, no bundler lock-in, nothing to unlearn.

Nothing to ship but standards

Components compile to plain Custom Elements. The runtime is one small ES module — no virtual DOM, no dependencies, no lock-in.

Light DOM, scoped CSS

Your page's stylesheet just works. Component CSS is scoped under its own tag automatically — Shadow DOM is an opt-in, not a default.

Reactivity you can read

Directives sit visibly in the markup and every expression is plain JavaScript. Updates flow from parent to child, and you can trigger them from anywhere.
