Ranges#

Sliders bind as numbers — drag it or set it, the property stays a number either way.


            <input type='range' max='11' lw-model='volume'>
            <b lw>volume</b>
            <button lw-on:click='volume = 11'>These go to eleven</button>
         

            class extends LWElement {
               volume = 7;
            }
