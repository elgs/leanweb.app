Checkboxes#

Bind a group of checkboxes to an array — checked values appear in it, unchecked ones leave. A single checkbox bound to a boolean just toggles it.


            <label lw-for='t in toppings'>
               <input type='checkbox' lw-bind:value='t' lw-model='chosen'>
               <span lw>t</span>
            </label>
            <button lw-on:click='chosen = [...toppings]'>All</button>
            <button lw-on:click='chosen = []'>None</button>
            <p lw>'Pizza with: ' + chosen.join(', ')</p>

            <label><input type='checkbox' lw-model='extraCheese'> extra cheese</label>
            <p lw>extraCheese ? 'Double it. 🧀' : 'Keeping it light.'</p>
         

            class extends LWElement {
               toppings = ['olives', 'feta', 'basil'];
               chosen = ['basil'];
               extraCheese = false;
            }

'Pizza with: ' + chosen.join(', ')

extraCheese ? 'Double it. 🧀' : 'Keeping it light.'
