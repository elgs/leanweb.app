$ lw init

Create a ready-to-go web component project in an empty directory. Uses light DOM with scoped CSS by default. Add --shadow-dom to enable Shadow DOM encapsulation.

$ lw serve

Start a dev server and watch for source code changes.

$ lw dist

Build your project for production.

$ lw upgrade

Upgrade the project runtime after updating Leanweb tools.

$ lw upgrade --diff

Dry run: show per-file diffs of what an upgrade would change in the project runtime — including any local modifications it would overwrite — without writing anything.

$ lw help

Get help information for all available commands.
