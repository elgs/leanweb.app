lw-class:

Toggle CSS classes dynamically with lw-class:. The class is applied when the bound expression returns true, and removed when it returns false.

HTML 

    <div lw 
      lw-for='item, $index in items' 
      lw-class:active='isActive($index)' 
      lw-on:click='setActive($index)'>
      item
    </div>
CSS 

   .active {
      color: red;
   }
Javascript 

   class extends LWElement {
      items = ['one', 'two', 'three'];
      setActive(index) {
        this.activeIndex = index;
      }
      isActive(index) {
        return index === (this.activeIndex ?? 1);
      }
   }
