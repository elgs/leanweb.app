lw-for

HTML 

   <div lw lw-for='item, $index in items'>$index+': '+item</div>
Javascript 

   class extends LWElement {
      items = ['one', 'two', 'three'];
   }
$index+': '+item

You could access DOM nodes for each element in a lw-for loop by calling elem.getDom() as long as typeof elem evaluates object.
