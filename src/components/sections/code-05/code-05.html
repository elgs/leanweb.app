lw-bind:

Dynamically set any HTML attribute with lw-bind:. Bind src, width, title, or any other attribute to a component property.

HTML 

   <img lw-bind:src='imgSrc' lw-bind:width='imageWidth' lw-bind:title='title'>
Javascript 

   class extends LWElement {
      imgSrc = '/resources/images/arizona.jpg';
      imageWidth = 400;
      title = 'Beautiful Arizona';
   }
