lw-if#

Render an element only while an expression is truthy. When it turns false the element is parked — not destroyed — and returns to its exact spot, state intact, when the expression turns true again.


            <button lw-on:click='on = !on'>Toggle</button>
            <div lw-if='on'>
               🎈 Parked, not destroyed — <input placeholder='type, then toggle twice'>
            </div>
         

            class extends LWElement {
               on = true;
            }
🎈 Parked, not destroyed —
