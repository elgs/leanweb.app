lw-bind:#

Set any HTML attribute from an expression. When the expression yields false, null, or undefined, the attribute is removed — which makes lw-bind:disabled and friends just work.


            <input type='range' min='140' max='320' lw-model='width'>
            <img lw-bind:src='src' lw-bind:width='width'
                 lw-bind:title='width + " pixels of Arizona"'>
         

            class extends LWElement {
               src = 'resources/images/arizona.jpg';
               width = 240;
            }
