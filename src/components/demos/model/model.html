lw-model#

Two-way binding: typing updates the property, and setting the property updates the control. It works on every form element — the Forms section below covers each kind.


            <input lw-model='city' placeholder='type a city'>
            <button lw-on:click="city = 'Phoenix'">Set Phoenix</button>
            <p>Shipping to <b lw>city</b></p>
         

            class extends LWElement {
               city = '';
            }
