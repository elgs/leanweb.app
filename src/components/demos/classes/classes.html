lw-class:#

Toggle a CSS class from a boolean expression — added while it's true, removed when it's false.


            <div class='row'>
               <button lw lw-for='size, $i in sizes'
                  lw-class:chosen='picked === $i'
                  lw-on:click='picked = $i'>size</button>
            </div>
         

            .chosen {
               outline: 2px solid #34d399;
            }
         

            class extends LWElement {
               sizes = ['Small', 'Medium', 'Large'];
               picked = 1;
            }
